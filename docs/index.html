Open Reliability Editor

Freedom of Research, Privacy of Data.

The Open Reliability Editor (ORE) is a free web-based tool for dependenability modeling and analysis.

ORE is the successor for the FuzzEd editor project, which was originally available at fuzzed.org. The old site is currently deactivated, while we are migrating code and data to the new home in the Open Reliability initiative. If you are interested in the migration progress, or need an emergency access to your old models, please talk to us. Nothing is lost.

Dependability Analysis in the Cloud.

ORE gives you the fastest way for creating system dependability models. Share your models and ideas with other experts, but only if you want to. We support static fault trees, dynamic fault trees, reliability block diagrams and FuzzTrees.

Make Uncertainties Explicit.

Our novel FuzzTree concept for fault tree modeling with uncertainties allows you to express configurable and currently underspecified parts of the system. Our analysis engine tells you which configuration options will give the best cost-dependability ratio.
