Dependability Modeling with Fault Trees and RBDs

If you need a quick way for drawing and analyzing a system reliability model, this one is for you. We offer a free basic account so that you can just start.

Modeling with uncertainties - Introducing FuzzTrees

Our novel concept for fault tree modeling allows you to express configurable and uncertain parts of the system in a reliability model. Our analysis engine tells you which configuration options give the best cost-dependability ratio.

Graph sharing for teaching and collaboration

Graphs can be easily shared between users, which supports the usage for teaching and collaboration projects. We support the export as graphic, eMail attachment or project file.
